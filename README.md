## üì¶ Quick Start

### 1. Clone & Install
```bash
git clone https://github.com/git-dev-crs/Fliqq-Video-call.git
cd Fliqq-Video-call

# Install Backend Dependencies
cd Backend
npm install

# Install Frontend Dependencies
cd ../frontend
npm install
```

### 2. Configure Environment
Create a `.env` file in the **Backend** directory:
```env
PORT=8000
MONGO_URI=your_mongodb_connection_string
```

### 3. Run the App
**Start Backend:**
```bash
cd Backend
npm run dev
```

**Start Frontend:**
```bash
cd frontend
npm start
```
Visit `http://localhost:3000` to start calling!
